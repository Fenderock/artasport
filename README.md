# Современный дизайн сайта ARTA Sport

В папке есть два варианта прототипа:

- `index.html` — рабочий локальный прототип на HTML/CSS/JS/jQuery.
- `clean.html` — чистая HTML-структура без CSS и JS.

React/Vite-экспорт из Figma сохранен в `src`, но для локального просмотра без установки зависимостей используйте `index.html`.

## Запуск React-версии

```bash
npm i
npm run dev
```
